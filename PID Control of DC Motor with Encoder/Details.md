## *Motor pins*
| *Color* | *Function* |
| :---: | :---: |
| RED | Motor GND |
| BLACK | Motor VCC |
| GREEN | Encoder GND |
| BLUE | Encoder VCC |
| YELLOW | Encoder A Output |
| WHITE | Encoder B Output |
