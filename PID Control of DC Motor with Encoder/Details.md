## *Motor pins*
| *Color* | *Function* |
| :---: | :---: |
| RED | Motor VCC |
| BLACK | Motor GND |
| GREEN | Encoder GND |
| BLUE | Encoder VCC |
| YELLOW | Encoder A Output |
| WHITE | Encoder B Output |

## *PID Library that I use -->* [PID_v1](https://github.com/br3ttb/Arduino-PID-Library)
